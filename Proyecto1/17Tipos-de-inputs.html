Tipos de Inputs

Input Type:
Button--> se comporta igual que un botón <button> no envia formularios
color:--> se usa para especificar un color
date:--> se usa para introducir una fecha
datetime--> mas viejo que la rueda; está obsoleto
datetime-local--> Fecha y hora, se fresea con el firefox mejor no usarse
email--> se utiliza para introducir un Email
hidden--> campo oculto, puede contener un valor pero no se va a ver
month--> se utiliza para introducir un mes
number--> se utiliza para introducir valores numéricos
password--> se utiliza para introducir contraseñas
range--> se utiliza para establecer un rango
reset--> te resetea el formulario, y la vida misma
search--> se utiliza para las barras de búsqueda
submit--> se utiliza para enviar el formulario al mas allá
tel--> se utiliza para introducir numeros telefonicos
text--> valor por defecto que ya trai el input
time--> se utiliza para introducir una hora
url--> se utiliza para introducir URLS tambien conocidos como links o zeldas papu de papus
week--> se utiliza para introducir el numero de semana del año
--------------------------------------------------------------------------------
radio--> radio buton en java, permite elegir solo una opcion de entre varias
checkbox--> checkbox permite elegir varias opciones de entre varias, contrario a radio
select--> crea una lista de opciones donde podemos elegir una o varias opciones
cada opción irá dentro de una etiqueta <option> </option>
si tenemos muchas opciones podemos ordenarlas por categorias a través de
la etiqueta <optgroup> con el atributo label para nombrar la categoria
list hay que darle una lista con el atributo list y luego datalist con id igual al atributo list
--------------------------------------------------------------------------------
fieldset--> se utiliza para agrupar elementos dentro de un formulario
legend--> representa una etiqueta para el contenido del fieldset
file--> este input se utiliza para cargar archivos y enviarlos desde un formulario
meter--> representa un valor dentro de un rango conocido
progress--> representa el progreso de una tarea
textarea--> este es el text area clasico clasico clasico de java pero clasico

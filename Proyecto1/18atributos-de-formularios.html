Atributos de los formularios

autofocus --> enfocar automaticamente un campo de texto
placeholder--> da una lista de lo que el usuario tiene que introducir
required --> hace que un campo sea obligatorio
readonly --> hace que un campo sea solo de lectura
disabled --> desactiva el campo
min - max --> establece minimos y maximos para un campo numérico
minlenght - maxlenght --> establece el minimo y maximo de carácteres en un campo de texto
selected --> el equivalente a los checked en los radiobutton
-----------------------------------------------------------------------------------------------------------

Ejemplos de formularios con atributos

Autofocus; listo pa escribir

Placeholder; una pista para saber como rellenar

Required; Obligatorio, Sí o Sí

Read only; si manda el valor

Disabled; no manda el valor

Min - Max para numeros (14-99)

Minlenght - Maxlenght para letras (3-15)

Selected; aparecer una opcion seleccionada (Canadá)

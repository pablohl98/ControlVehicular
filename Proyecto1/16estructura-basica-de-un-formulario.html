Estructura Basica de un Formulario

La estructura basica se compone de 4 elementos:
form --> es la etiqueta que engloba nuestro formulario
label --> sirve para escribir el nombre del campo a rellenar debe tener el atributo for al cual
se le indica un id que l que hará será emparejar el label con su input correspondiente
input --> es la caja de texto clásica, clásica clásica, el unico atributo obligatorio que tiene es el name
button --> crea un botón que permitirá enviar el formulario
